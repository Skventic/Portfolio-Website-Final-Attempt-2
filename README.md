# Savannah Ventic Portfolio Website

This is a render-ready Quarto website template.

## How to use

1. Open `personal_website.Rproj` in RStudio.
2. Click **Build** > **Render Website**.
3. Open `_site/index.html` to preview the website.
4. Edit the `.qmd` files to customize the pages.

## Pages

- `index.qmd`: Home page
- `about.qmd`: About page
- `projects.qmd`: Project page
- `dashboard.qmd`: Dashboard page
- `presentation.qmd`: Presentation link page
- `presentation/slides.qmd`: RevealJS presentation source
- `resume.qmd`: Resume page
- `contact.qmd`: Contact page

No `renv`, `vitae`, or extra R package setup is required.
