# Create variables

price <- 100

  tax_rate <- 0.08

))

Q7.4. What is the difference between lapply and sapply functions. Which map functions are similar to them? Consider that you want to calculate the exponentials of three numbers. In this case, which function you should use and why? Type your answer as a comment?

  • lapply(): returns a list.

  • sapply(): tries to simplify the result into a vector or matrix.

  • In the map family:

    • map() is like lapply()
    • map_dbl(), map_chr(), etc. are like sapply() because they return simpler outputs

To calculate exponentials of numbers, I would use sapply() or map_dbl() because I want a numeric vector, not a list.

Q8.4. Note that there are six years from 2012 to 2017 that have a significant number of observations, conducive for drawing charts by year. Since you don’t want to copy and paste the code again and again, you want to create a function that allows you to use it to draw all the charts by simply entering “year” information into the function. You will want to show “year” information in the chart as a subtitle. Create such a function.

plot_ratio_year <- function(selected_year) {
  chart_data <- student_ratio |>
    filter(year == selected_year) |>
    filter(indicator == "Lower Secondary Education") |>
    dplyr::select(country, student_ratio) |>
    drop_na() |>
    arrange(student_ratio)
  
  chart_data <- chart_data |>
    slice(c(1:10, (nrow(chart_data) - 9):nrow(chart_data)))
  
  print(chart_data)
  
  ggplot(chart_data, aes(x = reorder(country, student_ratio), y = student_ratio)) +
    geom_col() +
    coord_flip() +
    labs(
      title = "Top and Bottom 10 Countries in Student-Teacher Ratio",
      subtitle = paste("Year:", selected_year),
      x = "Country",
      y = "Student-Teacher Ratio"
    )
}

library(tidyverse)

# Load built-in dataset
data("airquality")

# Create function
avg_ozone_by_month <- function(data) {
  
  data |>
    group_by(Month) |>
    nest() |>
    mutate(
      avg_ozone = map_dbl(data, ~ mean(.x$Ozone, na.rm = TRUE))
    ) |>
    select(Month, avg_ozone)
}
